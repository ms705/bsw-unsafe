# Example programs for Brown Systems Week

Two examples:
 - `use-after-free`: demonstrates how unsafe Rust allows for use-after-free bugs.
 - `string-fun`: experimentation program to illustrate how unsafety can be exposed or hidden in Rust.

Compile using:
```bash
rustc use-after-free.rs -o use-after-free
rustc string-fun.rs -o string-fun
```
